INSTALL -- How to
=========================

Python:
=========================

1. Clone the repo git clone git@github.com:anandjeyahar/onlinestats.git
2. cd to repo
3. Run python onlinestats.py --port=<desired-port>
4. That's you're ready to go.


Clojure:
============================

1. Install clojure and lein
2. lein new luminus <projectname>
